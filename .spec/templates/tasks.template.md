# Tasks Template

## Purpose

Break an approved implementation plan into small, reviewable execution steps.

## When to Use

Use this template immediately before implementation begins.

## Expected Sections

1. Title
2. Related Plan
3. Task List
4. Validation Checklist
5. Notes and Assumptions

## Completion Criteria

- Tasks are small and independently understandable.
- Validation steps are attached to the work.
- Dependencies between tasks are explicit.
- The list is ready to guide implementation and review.
